# Guion Lección 2 - Introducción a HTML

Objetivo: que el estudiante escriba su primera página usando `<header>`, `<main>`, `<section>` y `<footer>`.

Pasos sugeridos:
1. Abrir el archivo `starter-files/lesson2-starter.html`.
2. Explicar qué hace cada etiqueta: `<header>`, `<main>`, `<section>`, `<footer>`.
3. Completar en vivo el `<h1>` y el primer `<p>` como ejemplo.
4. Pedir a los estudiantes que completen el resto de los TODO por su cuenta.
5. Comparar el resultado con `completed-examples/lesson2.html`.
